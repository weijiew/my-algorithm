# A+B for Input-Output Practice (II) HDU - 1090 

https://vjudge.net/problem/HDU-1090

用 cin/cout 太慢。

```cpp
#include <bits/stdc++.h>
using namespace std;
int main() {
    int a, b, c;
    scanf("%d", &a);
    while (a--) {
        scanf("%d %d", &b, &c);
        printf("%d\n",b + c);
    }

    return 0;
}
```
